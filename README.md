# Sistema de Gestão de Estacionamento (Console)

Projeto desenvolvido em **Java (Java Core)** com o objetivo de praticar os principais conceitos da linguagem, incluindo **Programação Orientada a Objetos**, **interfaces**, **herança**, **polimorfismo** e **arquitetura em camadas**, utilizando uma aplicação **via console**.

---

## Objetivo do Projeto

Simular o funcionamento de um estacionamento, permitindo:

- Registrar a **entrada de veículos**
- Registrar a **saída de veículos**
- Calcular o **valor a pagar** conforme o tempo estacionado
- Listar veículos atualmente estacionados
- Controlar o **faturamento total**
- Operar tudo através de um **menu interativo no console**

---

## Conceitos de Java Aplicados

- Java Core  
- Programação Orientada a Objetos (OOP)  
- Classes abstratas  
- Interfaces  
- Herança e Polimorfismo  
- Collections (`List`, `ArrayList`)  
- Estruturas de controle (`if`, `switch`, loops)  
- Organização em camadas (Model, Repository, Service)


---

## Como Executar

1. Abra o projeto em uma IDE Java (IntelliJ IDEA recomendado)
2. Execute a classe `Main`
3. Interaja com o menu pelo console
